Projects

Project 1: Facebook Marketplace Product Recommendation

Developed a product recommendation system inspired by Facebook Marketplace's approach to providing users with relevant suggestions. The PyTorch model was trained on a dataset of 12,000 product images, and leverages the power of FAISS to efficiently identify and display images similar to those favoured by users. To ensure seamless integration and scalability, the system utilizes a RESTful API via FastAPI and is hosted within a custom Docker container on Amazon EC2.

Project 2: Anomaly Detection in Large Image Sets

Created a robust system capable of identifying anomalies within a collection of approximately 1 million unlabelled and highly similar images, intended for time-lapse video creation but with wide applicability to other large image datasets. This system employed Keras/Tensorflow to derive image embeddings, and utilized HDBSCAN Clustering to segregate images into different categories based on their similarity - including an outlier class for images that deviated from established norms. For efficiency, scalability, and adaptability at a large scale, the system incorporates Meta FAISS to approximate groupings for new images, thereby enhancing overall performance.
View project on GitHub

Project 3: Natural Language Processing for Stock Market Prediction

Investigated a variety of NLP techniques with the aim of predicting stock market movements based on daily news headlines. The project processed 1.2 million headlines collected over 20 years and compared them to the daily fluctuations in the NYSE aggregated shares portfolio. Headlines were analysed to extract keywords that could potentially influence stock prices, and sentiment analysis was applied. Ultimately, no robust model emerged from the study, so retirement on a fortune remains a distant dream.

Project 4: Rock-Paper-Scissors Hand Gesture Classification

Developed a machine learning model to classify hand gestures in a live web-cam feed for playing a game of rock-paper-scissors. The model was trained on over 10,000 images achieving an accuracy of 90% on test data, and classified the new frames using OpenCV and TensorFlow.
